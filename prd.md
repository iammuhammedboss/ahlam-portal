PRD: Ahlam Dhofar Logistics Enquiry Portal1. Executive SummaryA streamlined web portal for Ahlam Dhofar Logistics to centralize customer enquiries. It replaces fragmented WhatsApp/Email communication with a single, secure "Chat Hub" managed by one primary agent.2. User Roles & AccessRoleAccess MethodPermissionsThe Agent (The "One Guy")Username + PasswordFull dashboard, manages all enquiries, sends files/invoices.The CustomerPhone/Enquiry ID + Unique PINViews their specific enquiry, sends messages, uploads files.3. The Customer Journey (Step-by-Step)Phase 1: IntakeLink Sharing: The agent sends a generic "Start Enquiry" link to the customer via WhatsApp.Blank Form: The customer clicks the link and fills in:Full Name, Phone Number, Email.Enquiry Message (Logistics requirements).File Upload: (Optional) Attach Packing Lists, ID copies, or photos.Submission: Customer hits "Submit."Phase 2: Secure Access GenerationUnique PIN: The system instantly generates a random, non-repeating 4-digit PIN (e.g., 9421).On-Screen Success: The screen displays:"Enquiry #1024 Submitted! Your Unique PIN is 9421. Please save this to log back in."Automated Email: no-reply@ahlamdhofar.com sends the PIN and a direct login link to the customer.4. The Agent Dashboard (The "Command Center")Designed for the "one guy" to follow up and manage everything in one place.Master Inbox & SortingThe List View: A table showing all active enquiries.Smart Sorting:Unread First: Any enquiry with a new customer message jumps to the top.Date/Time: Sort from newest to oldest.Status: Filter by New, Quoted, Pending Customer, or Closed.Search: Quickly find an enquiry by Phone Number or Name.The Interactive Chat WindowChat Interface: A real-time message thread (WhatsApp-style).Document Attachment: The agent can upload the Logistics Quote or the final Invoice.Invoice Tagging: When an invoice is uploaded, it is highlighted so the customer can find it instantly at the top of their screen.5. Technical Requirements (The "Lean" Stack)Domain: Subdomain management via your Squarespace (e.g., portal.ahlamdhofar.com).Mailing: Integrated SMTP for no-reply@ahlamdhofar.com.Database: Securely stores Enquiry Data, Unique PINs, and Message History.Responsiveness: Must work perfectly on mobile (for customers on WhatsApp) and desktop (for the agent).6. Notification LogicCustomer Alerts: Email notification whenever the agent sends a message or uploads an invoice.Agent Alerts: Email notification whenever a customer fills a new form or replies to an existing one.Final Prototype Checklist:[ ] Blank Form ready for customer input.[ ] Unique PIN generator (Random 4-digits).[ ] Automated "No-Reply" Email setup.[ ] Agent Dashboard with full sorting and file/invoice upload.[ ] Login Page for returning customers (Phone + PIN).
